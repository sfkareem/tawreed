# Spec: Output Refinement & Language Compliance

Improve the Excel, CSV, and PDF outputs generated by Tawreed to comply with strict language options, remove unnecessary supplier pricing columns, convert the brand column into an AI-extracted field, and ensure clean cell text wrapping across all worksheets.

## Proposed Changes

### 1. Language Compliance (English / Arabic / Bilingual)
* Update system prompts in `tawreed_backend.py` to enforce strict output language.
  * `"english"`: Extract/translate entirely to English. No Arabic characters.
  * `"arabic"`: Extract/translate entirely to Arabic. No English characters (except standard abbreviations/technical codes).
* Replace hardcoded bilingual titles, header labels, metadata strings, and helper text in the generated Excel sheets with dynamically localized strings matching the chosen language config.

### 2. Output Column Restructuring
* **JSON Schema (Prompts)**:
  * Remove `supplier_unit_rate`, `supplier_total_price`, and `supplier_remarks` from the prompt's output json definition.
  * Rename `supplier_brand` to `brand_model` (English/Arabic) or `brand_model_en`/`brand_model_ar` (Bilingual).
  * Direct the AI to only populate this field if the brand or model is explicitly stated in the source BOQ document; otherwise, it must return null or an empty string.
* **Excel Exporter**:
  * Remove column mappings for `supplier_unit_rate`, `supplier_total_price`, and `supplier_remarks`.
  * Rename the brand column to `"Brand / Model"` (or localized equivalents) and change its layout properties to display the AI's extracted value (read-only, no longer blank).
* **CSV/PDF Exporters**:
  * Update CSV and PDF layouts to include the new `brand_model` field.

### 3. Text wrapping and Formatting
* Enable `wrap_text=True` in `Alignment` configurations for all cells in:
  * Package Material sheets
  * `Summary` sheet
  * `Review Flags` sheet
  * `Assumptions` sheet
  * `Warnings` sheet
* Adjust row heights dynamically or allow auto-fitting.

## Verification Plan

### Manual Verification
* Run Tawreed with a sample BOQ.
* Test generating output in "English" and confirm that all generated sheets, summaries, metadata, warnings, and extracted values are in English only.
* Open the generated Excel file to verify that the three supplier columns are gone, the "Brand / Model" column is populated with extracted values, and the cell contents wrap correctly.
* Verify CSV and PDF export outputs.
